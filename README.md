# ArtThread

> Find the invisible threads linking art across time

## Deploy to Vercel

### Step 1 — Upload to GitHub
1. Go to github.com and click the **+** button → **New repository**
2. Name it `artthread`
3. Click **Create repository**
4. Upload all these files by dragging them in

### Step 2 — Deploy on Vercel
1. Go to vercel.com
2. Click **Add New Project**
3. Select your `artthread` GitHub repository
4. Click **Deploy**

### Step 3 — Add your API key
1. In Vercel, go to your project → **Settings** → **Environment Variables**
2. Add:
   - Name: `ANTHROPIC_API_KEY`
   - Value: your API key (the one you saved)
3. Click **Save**
4. Go to **Deployments** and click **Redeploy**

### Step 4 — Connect artthread.io
1. In Vercel, go to your project → **Settings** → **Domains**
2. Type `artthread.io` and click **Add**
3. Vercel will show you DNS settings to add in Namecheap/Cloudflare
4. Follow those instructions — takes ~10 minutes to go live
